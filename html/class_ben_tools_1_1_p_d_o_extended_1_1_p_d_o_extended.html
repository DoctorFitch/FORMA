sqlAssoc (   $sqlString,
  $sqlValues = array(),
  $dataType = self::TO_STRING 
)

sqlAssoc executes Query : If $dataType == self::TO_STRING : returns an associative array where the 1st column is the key and the 2nd is the value If $dataType == self::TO_STDCLASS : returns an associative array where the 1st column is the key the others are properties of an anonymous object If $dataType == self::TO_ARRAY_ASSOC : returns an associative array where the 1st column is the key the others are an associative array If $dataType == self::TO_ARRAY_INDEX : returns an associative array where the 1st column is the key the others are an indexed array

Parameters
